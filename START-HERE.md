# Roux by Melsplains
### Sunday Meal Planning Kit

Hi. Here's everything you need to stop spending Sunday evenings inside your own head.

The short version: you fill in a few files about your family, paste a prompt into Claude, and get back a real meal plan for your actual week — plus a grocery list of only what you need to buy.

> **New here? Start with the Claude.ai version.**
> You do not need AnyList. You do not need Claude Code. You do not need to install anything or know anything technical. Just a browser and a free Claude.ai account.

Two versions are included. Pick one based on how you work.

---

## Which version is for you?

### → Start Here — Claude.ai Version

Works in your browser. No extra apps, no installation, no command line. You fill in a few files about your family, attach them to Claude, paste the prompt. That's the whole thing.

**What you need:** A Claude.ai account. Free works.

**What you do NOT need:** AnyList · Claude Code · MCP setup · any coding knowledge · any subscription to try it

Claude Pro ($20/month) is worth it if you do this weekly — it unlocks Projects, which means your family preferences stay saved permanently. But try the free tier first to make sure this system works for you before paying for anything.

**Your files:** Everything in the `simplified/` folder.
**Setup guide:** `setup-guides/simplified-setup-guide.md`

**Time:** About 20 minutes the first time. 15 minutes every Sunday after that.

---

### Advanced — Claude Code + AnyList

This is the version where Claude browses your actual recipe library in AnyList, builds your week, and pushes the plan right into AnyList's calendar with the recipes attached. It requires a little setup — installing Claude Code on your computer and connecting it to AnyList. Not hard, but more steps.

**What you need:** AnyList (free to download, paid subscription recommended), Claude Code installed on your computer.

**Your files:** Everything in the `full-system/` folder.
**Setup guide:** `setup-guides/full-system-setup-guide.md`

**Time:** 30–60 minutes to set up the first time. 15 minutes every Sunday after that.

---

## About the first setup

The first time takes longer than it sounds — not because the tool is complicated, but because you're writing down things that have never been written down. Your family's food rules. The meals everyone actually likes. Which nights are impossible. Most people find that part surprisingly useful on its own. You're not entering data — you're offloading something that's been living in your head.

After the first week: 15 minutes. Every Sunday.

## What's in the Kit

| File | What it does | When you update it |
|------|-------------|-------------------|
| `simplified/01-family-preferences.md` | Who eats what, hard nos, protein rules, your usual meal rotation | Once (update when something changes) |
| `simplified/02-weekly-schedule.md` | This week: busy nights, weather, anything to use up | Every Sunday, ~3 min |
| `simplified/03-inventory.md` | What's in the freezer and pantry | As you cook; at minimum each Sunday |
| `simplified/04-sunday-planning-prompt.md` | The prompt you paste into Claude | Every Sunday |
| `simplified/05-grocery-list-format.md` | Reference for what Claude's output will look like | Read once |
| `full-system/` | Files for the AnyList + Claude Code version | See setup guide |
| `setup-guides/` | Step-by-step setup for both versions | Read once |

## Your Sunday in 15 Minutes

1. Update inventory (what you used this week) — 2 min
2. Fill in this week's schedule (busy nights, weather, freezer guilt items) — 3 min
3. Paste the prompt into Claude — 5 min
4. Glance at the plan, maybe swap one thing — 5 min
5. Copy the grocery list to your phone and go

---

## A Note on the Templates

Everything in the template files is a placeholder. The system works based on what *you* put in it. If the example family has three kids and you have one, change it. If you don't eat beef, change it. The more specific you are, the better the plan.

Nothing here is precious. Make it yours.
